# Homey App Onboarding

Learning kit for people joining the Homey app development team.

By the end of ~4 weeks you should be able to read the [Homey Apps SDK](https://apps.developer.homey.app) fluently, scaffold an app from scratch, build drivers, Flow cards and widgets, validate and publish — and explain what you built.

## What's in this repo

| File                                                  | What it is                                                                  |
| ----------------------------------------------------- | --------------------------------------------------------------------------- |
| [`onboarding-kit.md`](./01-onboarding-kit.md)            | The 4-week learning path with reading, videos, exercises and demos          |
| [`assignment.md`](./02-assignment.md)                    | The week-4 assignment spec (`weatherbuddy`) and acceptance criteria         |
| [`cheat-sheet.md`](./03-cheat-sheet.md)                  | One-page reference: CLI commands, file structure, common patterns           |

## Suggested reading order

1. Start with [`onboarding-kit.md`](./01-onboarding-kit.md) — read the intro, do the week-1 setup, follow the weekly path.
2. Keep [`cheat-sheet.md`](./03-cheat-sheet.md) open in a browser tab from day 1.
3. Open [`assignment.md`](./02-assignment.md) at the start of week 4.

## How to submit your assignment

You'll push your completed assignment to this repo as a PR. See the [submission instructions in `assignment.md`](./02-assignment.md#how-to-submit) for the step-by-step.

## Feedback

If anything is outdated, unclear, or missing — open an issue or a PR. This kit improves every time someone goes through it.
